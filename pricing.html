What's Included & Not Included

Typically Included

  • ✓ Accommodation in selected room type
  • ✓ Three Ayurvedic meals daily
  • ✓ Physician consultations (as prescribed)
  • ✓ Treatment procedures (as prescribed)
  • ✓ Morning yoga and chanting sessions
  • ✓ Evening satsang
  • ✓ Medications (as prescribed)
  • ✓ Yoga nidra sessions
  • ✓ Ayurveda cooking classes
  • ✓ Weekly community dinners

Typically Not Included

  • ✗ Travel to Coimbatore
  • ✗ Airport / station transfers (can be arranged)
  • ✗ Personal expenses
  • ✗ Visa and travel documentation
  • ✗ Laundry services

  • ✗ Take-home medications
  • ✗ External specialist consultations
